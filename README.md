# Java 8 Features
This Repository included all the features added in Java 8 version

## Featured Added in Java 8 are:
1. Lambda Expressions
2. Functional Interfaces
   - java.util.function Package (Predicate, Function, Supplier, etc.)
3. Stream API
    - Collectors and Reduction Operations
    - Parallel Streams
    - Collectors Utility Class (Grouping, Partitioning)
4. Interfaces:
   - Default Methods in Interfaces
   - Static Methods in Interfaces
5. New Date and Time API (java.time)
   - New Date/Time Classes (LocalDate, LocalTime, LocalDateTime, ZonedDateTime)
6. Method References
7. Nashorn JavaScript Engine
8. CompletableFuture and java.util.concurrent enhancements
9. Base64 Encoding and Decoding API
10. Annotations on Types (Type Annotations)
11. Repeating Annotations
12. Improved Type Inference
13. New Optional Class
14. Files.lines() and Streams for I/O
15. Improved Concurrency Utilities
16. ForEach Method on Collections
17. Clock and Instant Classes
18. Period and Duration Classes
19. Unsigned Arithmetic Support
20. StringJoiner Class
21. Parallel Array Sorting (Arrays.parallelSort)
22. Improved HashMap Performance
23. Remove PermGen Space (Replaced by Metaspace)
24. New API for Bulk Data Operations
25. CompletableFuture Async Pipelines
26. Enhanced Security and SSL/TLS Updates
